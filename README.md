Description
My edits fit the required adjustment goals detailed in this weeks sidequest requirements - The users are now able to toggle the moon gravity using the "m" key and are able to access a speed boost with a 7 second cool down by using the "f" key on their keyboards.

Learning Goals

The goal is to demonstrates how to implement a toggle for moon gravity within the game environment. Allow users can switch between standard Earth gravity and reduced moon gravity in real time, allowing for different movement dynamics and gameplay experiences.

GenAI
GenAI was used in this side quest to help creating the adjustments to the code in order to toggle the gravity and speed.
